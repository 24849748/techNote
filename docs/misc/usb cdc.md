如果有RX，TX，CTS，RTS 四个信号，则多半是支持硬流控的UART；

如果有 RX，TX，CTS ，RTS ，DTR，DSR 六个信号的话，RS232标准的可能性比较大。

* DTR (Data Terminal Ready)
* DSR (Data Set Ready)
* RTS (Request To Send)
* CTS (Clear To Send)
